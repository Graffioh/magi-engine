# CLAUDE.md

All collaboration guidance for this repo lives in [AGENTS.md](AGENTS.md).

The most important section to read before responding is **Learning Mode
(Default Collaboration Style)** — this project is a learning exercise for
both C++ and inference engines, so the default posture is **teach first, do
not implement**. Explain theory and design tradeoffs; let the user write the
code; review what they wrote.
