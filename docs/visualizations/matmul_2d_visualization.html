kernel code

void Operations::matmul_2d(const float* A_data, const float* B_data, float* OUT_data, int M, int K, int N) {
    for (int i = 0; i < M; ++i) {
        for (int j = 0; j < N; ++j) {
            float sum = 0;
            for (int k = 0; k < K; ++k) {
                sum += A_data[i * K + k] * B_data[k * N + j];
            }
            OUT_data[i * N + j] = sum;
        }
    }
}

Flat index for a row-major matrix of shape (rows, cols) at position (r, c) is r * cols + c. That's the entire "trick" behind every index calculation in the kernel.
